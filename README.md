# 🚀 Circular Gallery (Magic Scroll)

A modern **3D circular scrolling gallery** built using React and WebGL (OGL).
This project showcases smooth animations, curved layouts, and interactive UI.

---

## 🌐 Live Demo

👉 https://projectmagicscroll.netlify.app/

(can be only viewed on pc)

---

## ✨ Features

* 🎯 Smooth 3D scrolling animation
* 🎨 Curved gallery layout
* ⚡ High-performance WebGL rendering
* 🖼️ Custom image support
* 📱 Responsive design
